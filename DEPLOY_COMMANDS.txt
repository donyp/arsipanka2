================================================================================
TROUBLESHOOTING
================================================================================

If deployment fails or app doesn't start:

1. Check Hugging Face Space logs for errors
2. Verify environment variables are set (SUPABASE_URL, JWT_SECRET, etc.)
3. Look for error messages in the startup logs
4. Check if port 7860 is available (error message will show if not)

Common fixes:
- Restart the Space (Settings > Restart this space)
- Check Supabase credentials are correct
- Ensure JWT_SECRET is set (generate new one if needed:
  node -e "console.log(require('crypto').randomBytes(32).toString('hex'))")
