═══════════════════════════════════════════════════════════════════
⚡ SUPER QUICK DEPLOYMENT (7 LANGKAH)
═══════════════════════════════════════════════════════════════════

1. Buka: https://huggingface.co/spaces/create
   - Pilih "Docker" sebagai SDK
   - Nama: pusat-arsip-anka
   - Klik Create

2. Setup Git (di local machine):
   git config user.email "your@email.com"
   git config user.name "Your Name"
   git remote add hf https://huggingface.co/spaces/USERNAME/pusat-arsip-anka

3. Deploy (Windows):
   double-click: deploy.bat

   Or Manual (semua platform):
   git add .
   git commit -m "Deploy: Pusat Arsip Anka"
   git push -u hf main

4. Tunggu build (~2-5 menit)
   - Buka Space page
   - Monitor "App logs"

5. Generate JWT Secret (jalankan di terminal):
   node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"

6. Set Environment Variables di HF Space Settings:
   SUPABASE_URL (dari Supabase dashboard)
   SUPABASE_SERVICE_ROLE_KEY (dari Supabase)
   JWT_SECRET (dari langkah 5)
   PORT = 7860
   NODE_ENV = production

7. Akses app:
   https://YOUR_USERNAME-pusat-arsip-anka.hf.space
